##StringBuffer类

多数情况下建议使用stringbuffer，因为其有速度优势而且可修改。
public class Test{
  public static void main(String args[]){
    StringBuffer sBuffer = new StringBuffer("菜鸟教程官网：");
    sBuffer.append("www");
    sBuffer.append(".runoob");
    sBuffer.append(".com");
    System.out.println(sBuffer);  
  }
}

结果为  “菜鸟教程官网：www.runoob.com
